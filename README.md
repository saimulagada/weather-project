Project Title

 Wind Forecast Monitoring App

Overview

 This project visualizes wind power generation and forecast data.
 It allows users to compare actual wind generation with forecast predictions and analyze forecast errors.

Features

--> Interactive line chart for wind generation

--> Forecast horizon filtering

--> Time range filtering

--> Forecast vs Actual comparison

--> Forecast error analysis using Python

Tech Stack

Frontend:

React (Vite)
TailwindCSS
Recharts
Analysis:
Python
Pandas
Matplotlib
Jupyter Notebook

How to Run Frontend
npm install
npm run dev
Analysis

Open the notebook:

jupyter notebook analysis.ipynb

The notebook includes:

Forecast error statistics

Error vs horizon analysis

Wind power reliability analysis
